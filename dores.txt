o bojetivo principal do projeto é criar uma analise de dados de clientes antigos(?) e novos, que
serão seguidos constantemente para uma analise de lucro ou pedra.

A analise deve conter um numero previsto de lucro que o cliente deve obter, e que se não for alcançado
será acionado um alerta para maior atenção com o individuo

clientes são considerados novos durante um periodo de trinta dias

Multipages será necessário. Uma boa ideia seria ter pelo menos uma três páginas na qual eu trabalharia cada parte da aplicação, uma pagina de input, outra de acompanhamento de dados
e uma terceira de visão geral dos clientes, onde eu poderia ter uma tabela com todos os dados. Dessa forma seria bem mais organizado

Implementar machine learning é uma ideia super interessante, só que surpresa, eu não faço a menor ideia de como colocar um modelo  de ia não supervisionado numa aplicação web
